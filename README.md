# Setup Oh My Zsh on Fedora 41

This repository contains a Bash script to automate the installation of:

- **Oh My Zsh**
- **Powerlevel10k** theme
- Plugins: `zsh-syntax-highlighting` and `zsh-autosuggestions`

## Requirements

- Fedora 41 or later
- Administrator (root) privileges
